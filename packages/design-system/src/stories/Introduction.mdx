# Bofit Design System

## 📌 프로젝트 개요

Bofit은 사용자 맞춤 보험 추천 서비스입니다.  
**디자인 시스템**은 일관성 있는 UI/UX를 제공하기 위해 개발되었습니다.

## 🔗 주요 링크

- [GitHub](https://github.com/team-bofit/bofit-client)

## 📦 구성 요소

- **Button** - 기본 버튼, 아이콘 버튼, 토글 버튼 (예시)

## 🎨 Design Tokens

- Color
- Font-style
- Typography
- Width

## 🧑🏻‍💻 협업 규칙

- 모든 컴포넌트는 `storybook`에 스토리 포함 필수
- 디자인 시스템 내 공통 스타일 준수
- 컴포넌트 테스트 권장
